Sources used

    Coordinates of places (CSV Instructions)

    Military base locations (KML Instructions)
CSV instructions
  1. Comma delimited data was downloaded manually
  2. Data was processed via Ruby script using FasterCSV gem
  3. Data was cleaned of spaces and extra dashes
  4. Data was inserted into sqlite3 database with columns in CSV mapped to database columns accordingly
API instructions
  1. Using ruby net/http library, data was pulled via HTTP in XML form
  2. Lat/Lon was extracted using Nokogiri library
  3. Cleaned up of extra spaces / characters
KML instructions
  1. Data was downloaded manually
  2. Processed using ruby nokogiri XML library
  3. Relation name, coordinates were extracted and inserted into the database
Scraping Instructions
  1. Specific target webpage was chosen manually as an entry point
  2. Forms were anylized to get a page with proper links of all necessary information
  3. Data was further processed to get rid of padded spacings and inserted into database
