2011-10-31 The Group

 

  • git clone https://github.com/RandomEtc/modestmaps-processing.git Then copy the modestmaps directory within to the Processing Libraries folder.
  • Get the SQLibrary from https://github.com/downloads/camillol/sqlibrary/SQLibrary-0.1.1.zip IMPORTANT! Do not get the one at http://bezier.de/processing/libs/sql/ - it is horribly slow with SQLite.

  • GET proxml http://creativecomputing.cc/p5libs/proxml/proxml.zip
